# Data Documentation

Due to file size constraints, the raw and processed datasets are not tracked in this repository. 

## Dataset: CAMELS
We use the Catchment Attributes and Meteorology for Large-sample Studies (CAMELS) dataset.

### How to Obtain the Data
1. Download the prepackaged CAMELS dataset from the official USGS/NCAR repository.
2. Extract the downloaded archive.
3. Place the raw meteorological and streamflow time-series files into the `data/raw/` directory.

### Directory Structure
* `data/raw/`: Contains the original, untouched CAMELS dataset files (e.g., precipitation, temperature, streamflow).
* `data/processed/`: Contains cleaned and transformed datasets ready for model training, generated by `src/data_preprocessing.py`.

### Features and Labels
* **Input Features:** Time series of precipitation, maximum temperature, minimum temperature, and previous day streamflow.
* **Target Label:** Next day streamflow.
